## I design, build, and validate detection systems — from architecture to alert.

Blue Team Practitioner building hands-on SOC detection pipelines and writing validated detection rules from real-world malicious traffic. Focused on **Detection Engineering** — not just using tools, but understanding *why* alerts fire and *how* to make them better.
